# Toyoda Gosei 25th Anniversary - Interactive Greeting Card 🎉

A minimalist, modern web-based greeting card created as a submission for the Toyoda Gosei 25th Anniversary creative contest. 

Instead of submitting a standard text document or static image, this project combines all contest categories (a new motto, an official wish, a poem, and visual design) into a single, interactive web experience.
